상품 상세 페이지 생성기

브랜드별 상품 상세 이미지를 빠르게 제작하고 다운로드할 수 있는 도구입니다.

브랜드 선택

아래에서 작업할 브랜드를 선택하세요.

아웃도어프로덕츠 → 오디너리홀리데이 → 유니버셜오버롤 →

사용 방법

1 브랜드 선택

상단 탭 또는 위 카드에서 작업할 브랜드를 선택합니다.

2 좌측 패널에서 내용 입력

상품명, 설명, 사이즈 정보 등을 입력합니다. + 버튼으로 이미지 슬롯을 추가하고 파일을 선택하세요.

3 우측 미리보기에서 이미지 클릭

우측 미리보기의 이미지를 직접 클릭해서 교체할 수도 있습니다.

4 이미지 다운로드

우측 하단 다운로드 버튼을 클릭합니다. 분할 수를 입력하면 여러 장으로 나눠 ZIP 파일로 받을 수 있습니다.

주요 기능

  • 실시간 미리보기 — 입력 즉시 우측에 반영됩니다.
  • 이미지 추가/삭제 — + 버튼으로 슬롯 추가, 호버 시 ✕ 버튼으로 삭제합니다.
  • 이미지 직접 클릭 교체 — 우측 미리보기의 이미지를 클릭하면 바로 파일 선택이 열립니다.
  • 분할 다운로드 — 세로가 긴 상세 이미지를 1~10장으로 나눠 다운로드합니다. 2장 이상이면 ZIP으로 제공됩니다.
  • 브랜드별 양식 — 각 브랜드의 상세 페이지 디자인 양식에 맞게 개별 페이지가 제공됩니다.
💡 Tip

이미지는 가로 800px 기준으로 렌더링됩니다. 세로 길이는 원본 비율에 따라 자동 조절되므로, 가로 800px 이상의 이미지를 사용하면 가장 선명한 결과물을 얻을 수 있습니다.
